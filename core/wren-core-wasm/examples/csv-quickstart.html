CSV Quickstart

Three patterns for registerCsv(): (1) standard CSV with inferred schema, (2) TSV with a custom delimiter, and (3) headerless CSV with an explicit Arrow schema. All three load real files from data/.

1. orders.csv — inferred schema

data/orders.csv · defaults
await engine.registerCsv('orders', csv);

2. products.tsv — tab delimiter

data/products.tsv · delimiter: "\t"
await engine.registerCsv('products', tsv, {
  delimiter: '\t',
});

3. region_targets.csv — no header, explicit schema

data/region_targets.csv · header: false · schema: [region, quarter, target]
await engine.registerCsv('region_targets', csv, {
  header: false,
  schema: [
    { name: 'region',  type: 'string' },
    { name: 'quarter', type: 'string' },
    { name: 'target',  type: 'int64'  },
  ],
});
