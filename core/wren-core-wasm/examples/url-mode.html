URL Mode Demo

Demonstrates loadMDL with a URL source — DataFusion reads remote Parquet via HTTP range requests. No registerParquet needed.

Setup

  1. Place .parquet files in examples/data/
  2. Start the server: node examples/serve.mjs
  3. Open this page at http://localhost:8787/examples/url-mode.html
