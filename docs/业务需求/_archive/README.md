这里存放的是 `docs/业务需求` 下**暂不作为当前 UI 导入主链路**的历史/辅助材料，保留目的是便于追溯，而不是作为正式输入源。

当前已归档内容主要包括：

- `knowledge-base/analysis-rules.md`
- `knowledge-base/sql-templates.md`
- `knowledge-base/_templates/`
- `knowledge-base/table-suggested-questions.generated.csv`
- `knowledge-base/table-suggested-questions.generated.json`

使用原则：

1. **UI 导入** 只以 `../knowledge-base/analysis-rules/` 和 `../knowledge-base/sql-templates/` 下的单文件为准。
2. 归档文件可参考、可复用，但不应作为当前导入流程的权威来源。
3. 如果后续重新启用某份归档材料，应先确认是否需要回迁到正式目录。
