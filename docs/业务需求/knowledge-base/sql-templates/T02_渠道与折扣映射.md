# T02 渠道与折扣映射

## 模板用途

查询渠道名称、渠道商、折扣比例，用于补充综合日报表等结果维度。

## 建议问题（可转为 sql_pair.question）

- 查询渠道名称、渠道商、折扣比例。
- 查询某平台下各渠道的折扣配置与渠道商信息。

## 核心表/模型

- channel
- report_channel_data_percent_config
- tenant_plat

## 参数

- tenant_plat_id
- channel_id
- channel_partner_id

## SQL 模板

```sql
SELECT
    c.tenant_id,
    c.tenant_plat_id,
    tp.name AS tenant_plat_name,
    c.id AS channel_id,
    c.name AS channel_name,
    c.channel_partner_id,
    c.channel_partner_username,
    c.rate_agent_id,
    c.rate_agent_username,
    COALESCE(r.percent, 100.0000) AS report_percent,
    ROUND(COALESCE(r.percent, 100.0000) / 100, 6) AS report_percent_ratio,
    CASE WHEN r.id IS NULL THEN 0 ELSE 1 END AS has_percent_config,
    r.modify_time AS percent_config_modify_time,
    c.status AS channel_status,
    c.create_time AS channel_create_time,
    c.modify_time AS channel_modify_time
FROM channel c
INNER JOIN tenant_plat tp
        ON tp.id = c.tenant_plat_id
       AND tp.deleted = 0
LEFT JOIN report_channel_data_percent_config r
       ON r.channel_id = c.id
      AND r.deleted = 0
WHERE c.deleted = 0
  AND c.status = 3
  AND tp.enabled = 1
  AND c.tenant_plat_id = :tenant_plat_id
  AND (:channel_id IS NULL OR c.id = :channel_id)
  AND (:channel_partner_id IS NULL OR c.channel_partner_id = :channel_partner_id)
ORDER BY c.id;
```

## 备注

- `channel_id`、`channel_partner_id` 可传 `NULL` 表示不过滤。
- 若渠道未配置折扣比例，则按 `100%` 输出，便于上层直接做比例折算。
- SQL 按 TiDB / MySQL 8 风格编写；导入前需在实际 runtime datasource 下做一次校验。
