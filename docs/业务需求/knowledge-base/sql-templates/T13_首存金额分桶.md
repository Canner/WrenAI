# T13 首存金额分桶

## 模板用途

按首存金额固定档位输出人数与占比。

## 建议问题（可转为 sql_pair.question）

- 按首存金额固定档位输出人数与占比

## 核心表/模型

- dwd_order_deposit

## 参数

- tenant_plat_id
- channel_id
- start_date
- end_date

## SQL 模板

```sql
WITH first_deposit AS (
    SELECT
        DATE(d.callback_time) AS first_deposit_date,
        d.channel_id,
        d.player_id,
        d.actual_amount
    FROM dwd_order_deposit d
    WHERE d.status = 2
      AND d.times = 1
      AND d.tenant_plat_id = :tenant_plat_id
      AND d.channel_id = :channel_id
      AND d.callback_time >= :start_date
      AND d.callback_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
bucketed AS (
    SELECT
        f.first_deposit_date,
        f.channel_id,
        f.player_id,
        CASE
            WHEN f.actual_amount = 10 THEN '10元'
            WHEN f.actual_amount = 20 THEN '20元'
            WHEN f.actual_amount = 30 THEN '30元'
            WHEN f.actual_amount = 50 THEN '50元'
            WHEN f.actual_amount = 100 THEN '100元'
            WHEN f.actual_amount = 200 THEN '200元'
            WHEN f.actual_amount = 300 THEN '300元'
            WHEN f.actual_amount = 400 THEN '400元'
            WHEN f.actual_amount = 500 THEN '500元'
            WHEN f.actual_amount = 1000 THEN '1000元'
            WHEN f.actual_amount = 2000 THEN '2000元'
            WHEN f.actual_amount > 2000 THEN '2000元以上'
            ELSE '其他'
        END AS bucket_name,
        CASE
            WHEN f.actual_amount = 10 THEN 1
            WHEN f.actual_amount = 20 THEN 2
            WHEN f.actual_amount = 30 THEN 3
            WHEN f.actual_amount = 50 THEN 4
            WHEN f.actual_amount = 100 THEN 5
            WHEN f.actual_amount = 200 THEN 6
            WHEN f.actual_amount = 300 THEN 7
            WHEN f.actual_amount = 400 THEN 8
            WHEN f.actual_amount = 500 THEN 9
            WHEN f.actual_amount = 1000 THEN 10
            WHEN f.actual_amount = 2000 THEN 11
            WHEN f.actual_amount > 2000 THEN 12
            ELSE 13
        END AS bucket_sort
    FROM first_deposit f
),
daily_total AS (
    SELECT
        first_deposit_date,
        channel_id,
        COUNT(*) AS total_first_deposit_user_count
    FROM first_deposit
    GROUP BY first_deposit_date, channel_id
),
daily_bucket AS (
    SELECT
        first_deposit_date,
        channel_id,
        bucket_name,
        bucket_sort,
        COUNT(*) AS bucket_user_count
    FROM bucketed
    GROUP BY first_deposit_date, channel_id, bucket_name, bucket_sort
)
SELECT
    b.first_deposit_date,
    b.channel_id,
    b.bucket_name,
    b.bucket_user_count,
    t.total_first_deposit_user_count,
    ROUND(b.bucket_user_count / NULLIF(t.total_first_deposit_user_count, 0), 6) AS bucket_ratio
FROM daily_bucket b
INNER JOIN daily_total t
        ON b.first_deposit_date = t.first_deposit_date
       AND b.channel_id = t.channel_id
ORDER BY b.first_deposit_date, b.channel_id, b.bucket_sort;
```
