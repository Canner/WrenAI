# T09 所有用户区间汇总

## 模板用途

统计某渠道在指定区间内全部用户/分层用户的存款、充提差、有效投注、输赢、投充比、杀率。

## 建议问题（可转为 sql_pair.question）

- 统计某渠道在指定区间内全部用户的投充比和杀率
- 统计某渠道 TOP3 用户的投充比和杀率
- 统计某渠道在指定区间内全部用户/分层用户的存款、充提差、有效投注、输赢、投充比、杀率

## 核心表/模型

- dwd_order_deposit
- dwd_order_withdrawal
- dwd_bet_order

## 参数

- tenant_plat_id
- channel_id
- start_date
- end_date
- user_segment
- top_n

## SQL 模板

```sql
WITH deposit_activity_users AS (
    SELECT DISTINCT d.player_id
    FROM dwd_order_deposit d
    WHERE d.status = 2
      AND d.tenant_plat_id = :tenant_plat_id
      AND d.channel_id = :channel_id
      AND d.callback_time >= :start_date
      AND d.callback_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
withdraw_activity_users AS (
    SELECT DISTINCT w.player_id
    FROM dwd_order_withdrawal w
    WHERE w.status = 3
      AND w.tenant_plat_id = :tenant_plat_id
      AND w.channel_id = :channel_id
      AND w.callback_time >= :start_date
      AND w.callback_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
bet_activity_users AS (
    SELECT DISTINCT b.player_id
    FROM dwd_bet_order b
    WHERE b.settle_status = 1
      AND b.tenant_plat_id = :tenant_plat_id
      AND b.channel_id = :channel_id
      AND b.settle_time >= :start_date
      AND b.settle_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
active_users AS (
    SELECT player_id FROM deposit_activity_users
    UNION
    SELECT player_id FROM withdraw_activity_users
    UNION
    SELECT player_id FROM bet_activity_users
),
bet_rank_base AS (
    SELECT
        au.player_id,
        COALESCE(SUM(b.valid_bet_amount), 0) AS total_valid_bet_amount
    FROM active_users au
    LEFT JOIN dwd_bet_order b
           ON b.player_id = au.player_id
          AND b.tenant_plat_id = :tenant_plat_id
          AND b.channel_id = :channel_id
          AND b.settle_status = 1
          AND b.settle_time >= :start_date
          AND b.settle_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
    GROUP BY au.player_id
),
ranked_users AS (
    SELECT
        br.player_id,
        br.total_valid_bet_amount,
        ROW_NUMBER() OVER (ORDER BY br.total_valid_bet_amount DESC, br.player_id) AS bet_rank
    FROM bet_rank_base br
),
selected_users AS (
    SELECT ru.player_id
    FROM ranked_users ru
    WHERE :user_segment = 'ALL'
       OR (:user_segment = 'TOPN' AND ru.bet_rank <= :top_n)
       OR (:user_segment = 'NON_TOPN' AND ru.bet_rank > :top_n)
),
deposit_summary AS (
    SELECT
        COUNT(DISTINCT d.player_id) AS deposit_user_count,
        SUM(d.actual_amount) AS deposit_amount
    FROM dwd_order_deposit d
    INNER JOIN selected_users su
            ON d.player_id = su.player_id
    WHERE d.status = 2
      AND d.tenant_plat_id = :tenant_plat_id
      AND d.channel_id = :channel_id
      AND d.callback_time >= :start_date
      AND d.callback_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
withdraw_summary AS (
    SELECT
        COUNT(DISTINCT w.player_id) AS withdrawal_user_count,
        SUM(w.act_amount) AS withdrawal_amount
    FROM dwd_order_withdrawal w
    INNER JOIN selected_users su
            ON w.player_id = su.player_id
    WHERE w.status = 3
      AND w.tenant_plat_id = :tenant_plat_id
      AND w.channel_id = :channel_id
      AND w.callback_time >= :start_date
      AND w.callback_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
),
bet_summary AS (
    SELECT
        COUNT(DISTINCT b.player_id) AS bet_user_count,
        SUM(b.valid_bet_amount) AS valid_bet_amount,
        SUM(b.win_loss_amount) AS win_loss_amount
    FROM dwd_bet_order b
    INNER JOIN selected_users su
            ON b.player_id = su.player_id
    WHERE b.settle_status = 1
      AND b.tenant_plat_id = :tenant_plat_id
      AND b.channel_id = :channel_id
      AND b.settle_time >= :start_date
      AND b.settle_time < DATE_ADD(:end_date, INTERVAL 1 DAY)
)
SELECT
    :tenant_plat_id AS tenant_plat_id,
    :channel_id AS channel_id,
    :start_date AS start_date,
    :end_date AS end_date,
    :user_segment AS user_segment,
    (SELECT COUNT(*) FROM selected_users) AS user_count,
    COALESCE(d.deposit_user_count, 0) AS deposit_user_count,
    COALESCE(d.deposit_amount, 0) AS deposit_amount,
    COALESCE(w.withdrawal_user_count, 0) AS withdrawal_user_count,
    COALESCE(w.withdrawal_amount, 0) AS withdrawal_amount,
    COALESCE(d.deposit_amount, 0) - COALESCE(w.withdrawal_amount, 0) AS charge_withdraw_diff,
    COALESCE(b.bet_user_count, 0) AS bet_user_count,
    COALESCE(b.valid_bet_amount, 0) AS valid_bet_amount,
    COALESCE(b.win_loss_amount, 0) AS win_loss_amount,
    ROUND(COALESCE(b.win_loss_amount, 0) / NULLIF(COALESCE(b.valid_bet_amount, 0), 0), 6) AS kill_rate,
    ROUND(COALESCE(b.valid_bet_amount, 0) / NULLIF(COALESCE(d.deposit_amount, 0), 0), 6) AS bet_deposit_ratio
FROM deposit_summary d
CROSS JOIN withdraw_summary w
CROSS JOIN bet_summary b;
```

## 备注

- 可先做玩家周期汇总视图，再叠加 TOPN/VIP。
- 当 `:user_segment = 'ALL'` 时表示全部活跃用户；`TOPN` / `NON_TOPN` 依赖 `:top_n`。
- SQL 按 TiDB / MySQL 8 风格编写；导入前需在实际 runtime datasource 下做一次校验。
- 当前可视为 SQL 草案，校验通过后可转为 sql_pair。
